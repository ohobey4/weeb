머신러닝

데이터를 기반으로 학습하고, 예측하여 스스로 성능을 향상시키는 시스템과 알고리즘을 구축하는 기술이다. 알고리즘을 설계하기 어렵거나 프로그래밍 하기 어려운 작업들을 해결하기 위해 사용된다. 머신러닝의 종류는 지도학습 비지도학습 강화학습으로 나뉜다.


지도학습

미리 정답 데이터(학습 훈련 데이터)를 제공한 후 거기에서 규칙과 패턴을 스스로 학습하도록 하는 방법으로 학습 훈련 데이터가 있어야 사용 가능하다.(최소 몇만 개 이상의 학습 훈련 데이터가 필요)
학습 훈련 데이터를 만들 수 있는 에시 1. 인간이 스스로 정답을 맞출 수 있는 경우 2. 애초에 정답이 부여된 데이터를 활용할 경우


비지도학습

학습 훈련 데이터를 제공하지 않고 시행하는 방법으로 정답을 알 수 없거나 정답이 없는 데이터를 사용할 때 사용하며 데이터가 갖춘 규칙성을 추출할 수 있다.


강화학습

특정 상태에서 다양한 행동을 평가한 후 더 좋은 행동을 스스로 학습하는 방법으로 바둑, 장기와 같은 게임이나 로봇 동작을 제어하는 과정에서 주로 사용한다.
학습 훈련 데이터를 제공하지는 않지만, 행동 선택지와 선택한 행동이 좋은지 아닌지를 판정하는 기준을 사람이 제공한다. 규칙이 정해져 있고 평가 기준이 명확하면 사용할 수 있지만, 규칙을 찾을 수 있는 문제는 처리하지 못한다.
